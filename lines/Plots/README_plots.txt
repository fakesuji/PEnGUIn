Plotting outputs appear here from:
line.pbs, from the PEnGUIn/lines directory which calculates lines and other analysis information, 
calculate_lines.py, from the PEnGUIn/lines directory which is ran by lines.pbs,
all notebooks and other line python scripts implementing plotting methods from functions_plots.py.

The output directory is as follows:
-Plots/RUN_NAME/ANALYSIS_NAME
-this naming method groups by simulation run, then by analysis run on said simulation
