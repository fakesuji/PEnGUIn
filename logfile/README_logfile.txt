Logfile outputs appear here from:
line.pbs, from the PEnGUIn/lines directory which calculates lines and other analysis information, 
autorun.pbs, from the PEnGUIn directory which runs the PEnGUIn code, 
continue.pbs, from the PEnGUIn directory which reruns the PEnGUIn code from saved frame, and
parameter_switch.pbs, from the PEnGUIn directory which changes all of the parameters in src.
